Págína web, que pondra a prueba y bajo estres de ciertas técnicas de hacking, SQL Injection, se pondran a prueba las técnicas de cifrado.

En Seguridad en Redes abordaremos temas y conceptos relacionados a la seguridad de redes, amenazas de red y politicas de seguridad. Como proyecto se sometera y brindara de mecanismos de seguridad para ciertos ataques.
